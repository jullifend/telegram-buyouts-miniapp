# Telegram Mini App для учета броней

Готовый MVP мини-приложения для Telegram-бота: человек открывает mini app, выбирает артикул, получает бронь на 50 минут, читает инструкцию, нажимает «Товар выкуплен», а в админке фиксируется заявка.

## Что уже настроено

- 8 артикулов-заглушек.
- По 5 мест на каждый артикул в день.
- Места автоматически обновляются каждый день после 00:00 по часовому поясу `APP_TIMEZONE`.
- Таймер брони на 50 минут.
- Автоосвобождение просроченных броней.
- Кнопка «Товар выкуплен».
- Финальное сообщение: человеку предлагается написать оператору, что товар выкуплен.
- Скрытая админка по `admin_key`.
- В админке можно менять артикулы, ссылки, лимиты, инструкцию и ник оператора.
- Фото можно загружать прямо с компьютера/телефона в админке, без сторонних ссылок.
- Над изображением каждого артикула показывается остаток: например, «Осталось 3 из 5».
- База хранится в локальном файле SQLite `buyouts.sqlite3`. Загруженные фото хранятся локально в папке `uploads`.
- Бот логирует сообщения, которые люди отправили именно боту.

## Важное ограничение

Бот не видит личные переписки пользователей с вами в Telegram. Он видит только:

1. действия внутри mini app: бронь, выполнение, отмена, время, артикул, Telegram ID, имя, username;
2. сообщения, которые пользователь отправил непосредственно этому боту.

## Переменные окружения

Скопируйте `.env.example` в `.env` для локального запуска или добавьте эти переменные в Render:

```env
BOT_TOKEN=токен_от_BotFather
WEBAPP_URL=https://your-app.onrender.com
ADMIN_KEY=любой_секрет_для_админки
WEBHOOK_SECRET=любой_длинный_секрет
RESERVATION_MINUTES=50
DAILY_LIMIT_PER_PRODUCT=5
OPERATOR_USERNAME=ваш_ник_без_@
APP_TIMEZONE=Europe/Moscow
DEV_MODE=0
DB_PATH=./buyouts.sqlite3
MAX_UPLOAD_BYTES=8388608
```

## Запуск на Render

Build Command:

```bash
pip install -r requirements.txt
```

Start Command:

```bash
uvicorn app:app --host 0.0.0.0 --port $PORT
```

После запуска откройте:

```text
https://ваша-ссылка.onrender.com/admin?admin_key=ваш_ADMIN_KEY
```

В админке нажмите «Подключить Telegram». После этого бот начнет присылать кнопку открытия mini app.

## Локальный запуск для проверки

```bash
pip install -r requirements.txt
cp .env.example .env
# В .env поставьте DEV_MODE=1
uvicorn app:app --reload
```

Откройте:

```text
http://127.0.0.1:8000
http://127.0.0.1:8000/admin?admin_key=change_this_admin_key
```


## Фото

В админке у каждого артикула есть кнопка загрузки фото. Поддерживаются JPG, PNG и WEBP. Максимальный размер по умолчанию - 8 МБ. После загрузки путь к фото автоматически записывается в поле `image_url`, и картинка показывается пользователям в mini app.


## Ежедневное обновление мест

Места не нужно очищать вручную каждый день. В базе у каждой заявки есть `day_key`. Когда наступает новый день после 00:00 по часовому поясу `APP_TIMEZONE`, приложение автоматически начинает считать остатки заново. Старые заявки остаются в базе как история, но не занимают места нового дня.

Для России обычно оставьте:

```env
APP_TIMEZONE=Europe/Moscow
```
